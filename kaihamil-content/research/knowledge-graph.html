About the Knowledge Graph

This is an initial visualization of the Kai Hamil content ecosystem using RDF triples. Each node represents an entity (post, framework, concept, person), and edges represent semantic relationships.

Entity Types

  • Frameworks (thinking tools - System)
  • Products (quizzes, toolkits - System)
  • Posts (essays, articles)
  • Wisdom (explorations - Self)
  • Traditions (philosophy sources - Self)
  • Research (investigations)
  • Concepts (ideas, patterns)
  • People (authors, researchers)

Relationship Types

  • evolvesTo (version progression)
  • implements (puts into practice)
  • deepDiveOf (extends/elaborates)
  • drawsFrom (wisdom source)
  • entryPointFor (gateway to)
  • relatedTo (semantic connection)
  • references (cites/mentions)

Use Cases

  • Content discovery
  • Related content surfacing
  • Gaps identification
  • RAG constraint
  • Agent memory structure
