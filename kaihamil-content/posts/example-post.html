Example Post

February 8, 2026

This is an example post template. To create a new post:

  1. Copy this file and rename it (e.g., my-new-post.html)
  2. Edit the title, date, and content
  3. Add a link to it in posts/index.html
  4. Deploy to Netlify

You can include images:

<img src="../images/your-image.jpg" alt="Description">

Or reference diagrams:

<a href="../diagrams/your-diagram.html">View diagram →</a>

Formatting Tips

  • Use <h2> for post title
  • Use <h3> for section headers
  • Wrap content in <article> tags
  • Add .date class for the date
