Sample Prompts

Property Listing: Luxury Home

Write a compelling property description for a luxury home with these details: - Address: [PROPERTY ADDRESS] - Price: $[PRICE] - Bedrooms: [#] - Bathrooms: [#] - Square footage: [#] - Key features: [LIST FEATURES] Make it evocative and emotional. Highlight the lifestyle, not just the features. Use sensory language. Include a compelling opening hook. End with a call to action for showings. Target affluent buyers.

Social Media: Market Update

Create a social media post about the current real estate market: - Location: [YOUR MARKET] - Key stat: [STATISTIC] - Trend: [TREND] Make it educational but digestible. Use emojis. Include a question to drive engagement. Keep it under 150 words.

Email: Welcome Sequence

Write a welcome email for new subscribers: - Your name: [YOUR NAME] - Your market: [LOCATION] - What they'll get: market updates, tips, listings Make it personal. Set expectations. Build rapport. Include one valuable tip immediately.
