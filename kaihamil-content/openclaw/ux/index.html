📐 Notation Standards

All UX flows use Mermaid.JS flowchart notation for consistency and maintainability. This allows:

  • Version control of diagrams as code
  • Easy updates without design tools
  • Consistent rendering across documentation
  • Interactive zoom/pan in HTML views

Future consideration: BPMN 2.0 for more complex process workflows with swim lanes and decision gateways.

🎨 Color Coding

  • Gold border — Entry points (Home, Tools Index)
  • Blue — Assessment flows (Quiz, Advanced Assessment)
  • Pink — Wisdom tools (Free Hour, Hour Apart)
  • Green — Framework tools (5 Filters, Problem Audit)
  • Light green — Completion states
