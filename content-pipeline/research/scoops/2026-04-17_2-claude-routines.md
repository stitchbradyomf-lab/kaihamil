# Claude Code Launches "Routines" - Repeatable Automated Workflows

## Summary
Launched alongside the desktop redesign on April 14, 2026, "Routines" brings repeatable automation to Claude Code, allowing users to save and replay multi-step workflows.

## What Are Routines?
- Save sequences of Claude interactions as reusable templates
- Automate repetitive coding tasks (refactoring, testing, documentation)
- Share routines across team members
- Trigger routines via CLI or the new desktop interface

## Key Use Cases
1. **Onboarding**: Automated repo setup for new developers
2. **Code Review**: Standardized review checklists
3. **Refactoring**: Pattern-based code transformations
4. **Testing**: Automated test generation and execution
5. **Documentation**: Generate docs from code changes

## Why This Matters
- Moves Claude Code from reactive assistant to proactive automation platform
- Reduces cognitive load for repetitive tasks
- Enables knowledge capture and sharing within teams
- Research preview suggests more features coming

## Competitive Context
- Similar to Cursor's "Composer" and GitHub Copilot's agent mode
- More focused on coding workflows than general automation
- Tighter integration with the development environment
