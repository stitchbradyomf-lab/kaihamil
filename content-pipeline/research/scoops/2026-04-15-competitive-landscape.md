## Windsurf (formerly Codeium)

### Acquisition History
- **December 2025:** Acquired by **Cognition AI** for ~$250M
- **Early 2025:** OpenAI reportedly acquired Codeium for $3B (conflicting reports - may refer to different assets)

### Positioning
- **Low-code, AI-native IDE** (not no-code - requires code understanding)
- VS Code fork with familiar extension ecosystem
- **Cascade:** Agentic engine built from scratch (not bolted on)

### Key Features
- Multi-model support: GPT-4o, Claude, Codeium's own models
- **Windsurf Tab:** Exclusive to Windsurf Editor - "single keystroke, limitless power"
- One-click setup for extensions/MCPs
- Runs pytest, pylint, radon in parallel - identifies issues within one second
- Deep contextual awareness for production codebases

### Pricing
- **Pro: $15/month** ($5 less than Cursor equivalent)
- Generous free tier
- Credits burn fast on some workflows

### Target Audience
- Professional devs tired of boilerplate
- Teams wanting consistent rules
- Startups
- Beginner-friendly while powerful for experienced devs

## Cursor

### Positioning
- Larger community than Windsurf
- More integrations
- Faster release cycle
- $20/month Pro tier

### Competitive Moat
- First-mover advantage in agentic coding
- Strong community/ecosystem
- Established plugin marketplace

## Market Dynamics
- **Consolidation happening:** Acquisitions (Cognition buying Windsurf)
- **Price competition:** Windsurf undercutting Cursor by $5/month
- **Feature parity:** All major players have agentic capabilities now
- **Model agnostic vs proprietary:** Windsurf offers multi-model; Cursor may be more opinionated

## Actionable Insights

- **For content:** "VS Code fork" is a familiar anchor for explaining new tools
- **For couples-vale:** $15-20/month price point established as market standard for pro tiers
