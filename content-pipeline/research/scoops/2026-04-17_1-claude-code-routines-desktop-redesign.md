## Summary

Anthropic released a dual update: a complete redesign of the Claude Code desktop app (Mac and Windows) with parallel session management, and the launch of "Routines" in research preview - a way to automate tasks without full agent autonomy.

## Key Details

- **Desktop Redesign**: New sidebar for managing multiple sessions, drag-and-drop workspace layout
- **Routines**: Automated task sequences without relying on fully autonomous agents
- **Parallel Sessions**: Run multiple Claude Code sessions simultaneously
- **Research Preview**: Routines currently in preview mode

## Implications

The "Routines" feature is interesting - it's a middle ground between fully manual and fully autonomous. Anthropic is recognizing that users want automation but with guardrails. The desktop redesign shows they're serious about competing with Cursor and other IDE-integrated agents.
