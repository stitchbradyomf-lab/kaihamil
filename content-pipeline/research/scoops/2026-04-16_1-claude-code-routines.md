# Claude Code Gets "Routines" - Automated Workflows Without Active Sessions

## Summary
Anthropic announced a major dual release on April 14, 2026: a complete redesign of the Claude Code desktop app and the launch of "Routines" in research preview. Routines allow users to set up Claude Code automations that run without requiring an active session.

## Key Details
- **Parallel Sessions**: New desktop app supports multiple Claude sessions side-by-side from one window with a sidebar to manage them all
- **Integrated Terminal**: Built-in terminal, side chat, improved diff viewer, and rearrangeable panes for agentic coding
- **Routines Feature**: Automations that can run on schedules or triggers without user actively monitoring
- **Availability**: Rolling out to Claude Code users on Pro, Max, Team, and Enterprise plans
- **Platform**: Mac and Windows desktop apps

## Why It Matters
This positions Claude Code as a true automation platform, not just an interactive coding assistant. The "Routines" feature directly competes with cron jobs, scheduled tasks, and workflow automation tools. For Vale OS and similar agentic platforms, this validates the market direction while also raising the competitive bar.
