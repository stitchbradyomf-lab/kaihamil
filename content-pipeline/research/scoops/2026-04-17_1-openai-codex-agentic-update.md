## Summary

OpenAI released a major Codex update adding computer use capabilities (Mac desktop control), built-in image generation, memory/personalization, and web browsing - positioning Codex as the foundation for OpenAI's upcoming "super app."

## Key Details

- **Computer Use**: Codex can now operate desktop Mac apps with its own cursor, clicking and typing
- **Multi-Agent**: Can run multiple agents in parallel without interfering with user work
- **Image Generation**: Built-in image generation capabilities
- **Memory**: Personalization and context-aware responses
- **Web Browser**: Integrated browsing capability
- **Super App Foundation**: These updates lay groundwork for OpenAI's rumored super app

## Implications

This is OpenAI's answer to Claude Code. The computer use feature is particularly significant - agents that can actually control your desktop. The "super app" narrative suggests OpenAI wants Codex to be more than just coding - a general-purpose agent workspace.
