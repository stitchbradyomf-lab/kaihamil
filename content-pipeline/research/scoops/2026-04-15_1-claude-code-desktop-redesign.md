# Claude Code Desktop Redesign: Integrated Terminal & Side Chat

**Date:** 2026-04-15
**Run:** 1
**Relevance Score:** 6/10
**Tags:** content, infrastructure

## Summary

Anthropic redesigned the **Claude Code desktop app** with features aimed at reducing context switching during agentic coding sessions.

## New Features

- **Integrated terminal:** No more switching between windows
- **Side chat:** Parallel conversations while main agent works
- **Improved diff viewer:** Better visualization of code changes
- **Rearrangeable panes:** Customize layout for workflow

## Platform Availability

Claude Code now available across:
- Terminal (original)
- VS Code extension
- JetBrains plugin
- Desktop app (redesigned)
- Web IDE at claude.ai/code

## Why This Matters

Desktop redesign reflects demand for "agent dashboard" UX - watching multiple things happen simultaneously. Similar to Cursor 3's Agents Window philosophy.
