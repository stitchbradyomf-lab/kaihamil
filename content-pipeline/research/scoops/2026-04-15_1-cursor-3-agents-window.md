# Cursor 3 Launches: AI Agents Window Redesign

**Date:** 2026-04-15
**Run:** 1
**Relevance Score:** 8/10
**Tags:** infrastructure, content

## Summary

Cursor launched version 3 on **April 2, 2026** with a major interface rebuild centered on a new **Agents Window** - reflecting the shift that "most code will be written by AI agents."

## Key Features

- **Agents Window:** Run multiple agents in parallel across local, cloud, worktree, and remote SSH environments
- **Design Mode:** Visual UI iteration - click on specific UI elements instead of describing them
- **Tiled Layout:** Split views for managing several agents simultaneously (v3.1 on April 13)
- **Cloud-to-local handoff:** Seamless transitions between cloud and local execution
- **Composer 2:** Their own frontier coding model running at 200+ tokens/second
- **Access:** `Cmd+Shift+P → Agents Window`

## Market Position

From recent comparisons:
- Cursor at $20/month vs Windsurf at $15/month
- Windsurf ranked #1 in LogRocket AI Dev Tool Power Rankings (Feb 2026)
- Windsurf now owned by Cognition AI (the Devin team)

## Why This Matters

Cursor betting everything on agent-first UX. This aligns with OpenClaw's vision - agents as first-class citizens, not bolted-on features. The parallel execution pattern is particularly relevant.
