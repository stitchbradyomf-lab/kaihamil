## Summary

Anthropic released Claude Opus 4.7 on April 16, 2026, positioning it as their most capable generally available model for complex reasoning and agentic coding. The release comes at the same pricing as Opus 4.6 ($5/$25 per MTok).

## Key Updates

- **Better coding capabilities**: Significant improvements in software engineering tasks
- **Triple the image resolution**: Enhanced vision capabilities compared to prior models
- **Self-checking ability**: New capability to double-check its own work
- **Auto mode expansion**: Setting where Claude can make autonomous decisions without constant permission prompts now extended to Max plan users
- **Cybersecurity safeguards**: Automated security protections built-in

## Strategic Context

- Anthropic concedes Opus 4.7 is "less broadly capable" than their unreleased Claude Mythos Preview
- This positions Opus 4.7 as the best generally available model while Mythos remains in preview
- Directly competes with OpenAI's recent GPT-5.4-Cyber release
