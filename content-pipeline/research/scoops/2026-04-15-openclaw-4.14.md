## Key Features

### v2026.4.14-beta.1 (Latest)
- **Telegram Forum Topics:** Human-readable topic names now appear in agent context and plugin metadata (previously abstract thread IDs only)
- **Browser/CDP Improvements:** Managed local Chrome readiness probes bypass SSRF policy for loopback control plane - fixes misclassification of healthy child browsers
- **QA/Testing:** New `--runner multipass` lane for OpenClaw QA suite - runs repo-backed scenarios in disposable Linux VMs with full report logging
- **Docs i18n:** Chunked translation, truncated output rejection, Pi session recovery

### Recent CLI Enhancements
- **`openclaw infer`**: First-class CLI inference hub for provider-backed workflows (model, media, web, embedding tasks)
- **Memory Wiki Reconstruction:** Full-stack restoration - plugins, CLI, toolchains, structured claims
- **Fast Mode for GPT-5.4:** Optimized for speed over deep reasoning
