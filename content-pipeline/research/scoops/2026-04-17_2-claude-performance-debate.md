# Claude Performance Debate: 67% Dumber? Developer Investigation

## Summary
A controversy emerged on Reddit r/ClaudeCode claiming Anthropic degraded Claude's performance without notice. The claim was backed by AMD's Director of AI, Stella Laurenzo, who analyzed 6,852 sessions.

## The Claims
- Claude reportedly made "67% dumber" responses
- Performance degradation in planning and complex reasoning tasks
- Coding quality remains mostly fine but with occasional "slop"
- Not following established code conventions consistently

## Community Response
- Mixed reactions: some confirm issues, others see no difference
- Debate over whether it's model degradation or perception bias
- Discussion of evaluation methodology and sample size
- GitHub issue filed: https://github.com/anthropics/claude-code/issues/42796

## Anthropic's Position
- No official response to the specific claims
- Recent releases (Opus 4.6, Sonnet 4.6) focus on output length (300k tokens)
- "Infinite Chats" feature eliminates context window limits
