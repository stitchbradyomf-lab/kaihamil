# 4 Elements of Healthy Vendor Management

Documenting the framework Kyle uses for vendor relationship management.

## The Elements

1. **Performance Metrics / Performance Reporting**
   - Regular reporting on agreed-upon KPIs
   - Visibility into service levels and outcomes

2. **Diagnostic Transparency**
   - Understanding root causes when performance issues arise
   - Open access to data and processes behind the metrics

3. **Improvement Actions**
   - Clear plans to address gaps or issues
   - Accountability for remediation
