# ClearPath Operations
## Implementation Guide (Internal Operations)

---

## Service Overview

**What we deliver:** A fully documented organizational system + ongoing maintenance for small service businesses.

**Core promise:** Client can find any document, see any project status, and know what needs attention—in under 30 seconds.

---

## Phase 1: Discovery & Audit (Week 1)

### Day 1-2: Chaos Mapping Call (90 min)
**Objective:** Understand current state and pain points

**Checklist:**
- [ ] Screen share: Walk through current file locations (all of them)
- [ ] Email tour: How many inboxes? What's the volume? What falls through?
- [ ] Document current tools in use (Google Drive, Dropbox, Notes, etc.)
- [ ] Identify communication channels (email, text, WhatsApp, Slack, etc.)
- [ ] List all client-facing touchpoints
- [ ] Ask: "What keeps you up at night organizationally?"
- [ ] Ask: "What's the last thing you couldn't find?"
- [ ] Ask: "If you had a magic wand, what would 'organized' look like?"

**Output:** Chaos Audit Document

### Day 3-5: Analysis & System Design
**Objective:** Design the custom Command Center

**Activities:**
- Categorize all information types (client files, projects, reference docs, etc.)
- Design folder architecture
- Select tools based on client's existing ecosystem
- Map current workflows → optimized workflows
- Create naming conventions
- Design tagging/labeling system
- Identify automation opportunities

**Output:** System Blueprint (shared with client for approval)

---

## Phase 2: System Build (Week 2)

### Day 1-2: Infrastructure Setup
**Objective:** Build the skeleton

**Checklist:**
- [ ] Create master folder structure
- [ ] Set up naming convention templates
- [ ] Configure selected tools (Notion, Google Workspace, etc.)
- [ ] Create client intake templates
- [ ] Create project tracking dashboard
- [ ] Set up automated folder creation (if applicable)
- [ ] Configure email filters/labels

### Day 3-4: Migration
**Objective:** Move existing chaos into new system

**Checklist:**
- [ ] Migrate and reorganize existing client files
- [ ] Process email backlog (sort, file, or archive)
- [ ] Consolidate scattered notes into central system
- [ ] Import contacts with proper tagging
- [ ] Document any "homeless" items that need client decision

### Day 5: Automation Layer
**Objective:** Make it self-maintaining

**Checklist:**
- [ ] Set up recurring task reminders
- [ ] Configure calendar integrations
- [ ] Create email-to-task automations (if applicable)
- [ ] Set up weekly review prompts
- [ ] Test all automations

---

## Phase 3: Training & Handoff (Week 3)

### Day 1-2: Training Sessions (2 x 60 min)
**Session 1: The System**
- Full walkthrough of Command Center
- Where everything lives
- How to find anything
- Naming conventions explained

**Session 2: Daily Habits**
- Morning routine (5 min check)
- End of day routine (5 min close)
- Weekly review process
- "What do I do when..." scenarios

### Day 3-4: Guided Practice
**Objective:** Client uses system with support

**Activities:**
- Client processes new items into system
- Real-time feedback via async video or chat
- Address any friction points
- Make tweaks based on actual usage

### Day 5: Documentation Delivery
**Deliverables:**
- [ ] System Manual (PDF or Notion doc)
- [ ] Quick Reference Card (1-pager for desk)
- [ ] Video walkthrough (recorded)
- [ ] "If Lost, Start Here" troubleshooting guide

---

## Phase 4: Ongoing Maintenance

### Weekly (30-60 min)
- Process inbox overflow
- File any homeless documents
- Update project statuses
- Flag items needing client attention

### Monthly (60-90 min)
- Full system review
- Archive completed projects
- Update any templates
- Check-in call: What's working? What's friction?
- Adjust system based on business changes

### Quarterly
- Deep audit of system usage
- Identify unused areas (why?)
- Propose optimizations
- Strategic planning: upcoming needs?

## Tool Stack (Default)

### Primary Options (choose based on client ecosystem):

**For Google Workspace users:**
- Google Drive (files)
- Google Calendar (scheduling)
- Google Tasks or Todoist (tasks)
- Google Contacts (CRM-lite)
- Gmail filters + labels

**For Apple ecosystem:**
- iCloud Drive or Dropbox (files)
- Apple Calendar
- Apple Reminders or Todoist
- Apple Contacts

**For "I want one tool":**
- Notion (all-in-one Command Center)
- or Coda
- or ClickUp (if more project-heavy)

**For CRM needs:**
- HubSpot Free (if client-heavy)
- Pipedrive (if sales-heavy)
- Folk (if relationship-focused)

**Automation:**
- Zapier (for cross-tool automations)
- IFTTT (simpler automations)
- Native integrations where possible

## Success Metrics

Track at 30/60/90 days:
- Time to find specific document (target: <30 sec)
- "I can't find it" incidents per week (target: 0)
- Missed follow-ups per month (target: 0)
- Client-reported stress level (1-10 scale)
- System actually being used (check activity logs)

---

## Red Flags & When to Fire a Client

- Refuses to spend 10 min/day maintaining habits
- Continues creating files outside the system
- Wants you to do VA work (respond to emails, etc.)
- Scope creep: "Can you also manage my social media?"
- Chronically reschedules training sessions

**Exit gracefully:** "This service works best for people who can commit to the habits. I'd recommend [VA service] for more hands-on support."
